========================================
  E-자산통합관리 - 실행 가이드
========================================

[ 1단계 ] Node.js 설치 (최초 1회)
---------------------------------------
1. https://nodejs.org 접속
2. LTS 버전 클릭 → 설치 파일 다운로드
3. 설치 파일 실행 → Next-Next-Finish
4. 설치 완료 후 PC 재시작 권장


[ 2단계 ] 이 폴더를 C:\IT-Manager 에 저장
---------------------------------------
- 이 폴더 전체를 C:\IT-Manager 로 복사


[ 3단계 ] 패키지 설치 (최초 1회)
---------------------------------------
1. 시작 → cmd (명령 프롬프트) 검색 → 실행
2. 아래 명령어 입력:

   cd C:\IT-Manager
   npm install

3. "added XX packages" 메시지 나오면 완료


[ 4단계 ] 서버 실행
---------------------------------------
cmd 에서:

   node app.js

아래 메시지가 나오면 정상:
   ====================================
    E-자산통합관리 구동 완료!
   ====================================
    브라우저에서 접속하세요:
    http://localhost:3000
   ====================================


[ 5단계 ] 브라우저에서 접속
---------------------------------------
Chrome 또는 Edge 주소창에 입력:

   http://localhost:3000


========================================
  자주 묻는 질문
========================================

Q. 서버를 끄려면?
A. cmd 창에서 Ctrl+C 누르면 종료

Q. 데이터는 어디에 저장되나?
A. C:\IT-Manager\it_manager.db (SQLite 파일)
   이 파일이 실제 데이터베이스입니다.
   백업할 때 이 파일을 복사하세요.

Q. 다른 PC에서 접속하려면?
A. 서버 PC의 IP 주소 확인 (ipconfig)
   다른 PC 브라우저에서: http://서버IP:3000

Q. 포트 3000이 사용 중이라고 나오면?
A. app.js 파일 2번째 줄의 PORT=3000 을
   다른 번호로 바꾸세요 (예: 3001, 8080)

Q. npm install 에서 오류가 나면?
A. Node.js 를 재설치하거나
   관리자 권한으로 cmd 실행 후 다시 시도

========================================
  서버(Windows Server)에 영구 설치 시
========================================

PM2 설치 후 자동 실행 등록:

   npm install -g pm2
   cd C:\IT-Manager
   pm2 start app.js --name "it-manager"
   pm2 startup
   pm2 save

이후 서버 재부팅해도 자동 실행됩니다.
